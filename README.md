# NFT Landing Page

This project is a modern, responsive NFT landing page featuring:
- Hero section with wallet connect
- NFT auction showcase
- Popular artists
- Community join section
- Trending NFTs grid
- Stylish footer

## Getting Started

### 1. Deploy on Vercel
1. Go to [vercel.com](https://vercel.com/) and sign in or create an account.
2. Click **New Project** and import this repository.
3. Vercel will auto-detect this as a static site (HTML/CSS).
4. Click **Deploy**. Your site will be live in seconds.

### 2. Local Development
1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Open the folder in VS Code or your editor.
3. Use a live server extension or open `index.html` in your browser.

## Project Structure
```
├── Images/           # All images and icons
├── index.html        # Main landing page
├── style.css         # Main stylesheet
├── README.md         # Project documentation
```

## Customization
- Replace images in the `Images/` folder for your own NFTs and avatars.
- Update text, links, and social icons in `index.html` as needed.
